# 功能介绍

## 快捷键支持

插件提供丰富的快捷键功能，让您的操作更加高效：

| 快捷键                   | 功能                  | 适用页面               | 默认状态 |
| ------------------------ | --------------------- | ---------------------- | -------- |
| `Ctrl + S`               | 保存检验单            | 样品检验主界面         | 可配置   |
| `Ctrl + D`               | 刷新列表/打开导入窗口 | 样品检验列表、初验页面 | 可配置   |
| 点击项目编号             | 复制项目编号          | 样品检验主界面         | 可配置   |
| 双击 `Ctrl`              | 复制项目编号          | 样品检验主界面         | 可配置   |
| 双击项目名称             | 复制项目中文名称      | 样品检验主界面         | 可配置   |
| `Ctrl` + 双击 `C` 键两次 | 复制项目中文名称      | 样品检验主界面         | 可配置   |
| 双击品名                 | 截图并复制到剪贴板    | 初验页面               | 可配置   |

### Ctrl + D 快捷键详细说明

`Ctrl + D` 在不同页面有不同功能：

- **初验页面列表**：刷新列表
- **样品检验主界面**：打开导入窗口
- **样品检验列表**：刷新列表
- **导入窗口**：自动搜索剪切板中的项目编号并导入

## 全局功能

### 1. 默认显示 100 条结果

- 自动将列表每页显示条数设置为 100
- 减少翻页次数，提高查询效率
- **默认开启**

### 2. 浏览器内实现所有功能

- 所有功能都在浏览器中实现，不依赖外部概要解析器
- 提高响应速度和稳定性
- **默认开启**

## 业务受理功能

### 1. 一键分配 / 一键回退

- **一键分配**：勾选检验单后，点击"一键分配"按钮直接将检验单分配给主检员
- **一键回退**：快速回退检验单到上一状态
- 支持批量操作，提高工作效率
- **默认开启**

### 2. 一键提交试验单

- 快速提交试验单，支持多种试验类型
- 自动识别试验项目并填充相关信息
- 支持的试验类型包括：
  - 气溶胶着火距离试验
  - 液体闪点试验
  - 易燃固体燃烧试验
  - 磁性物质试验
  - 等多种试验
- **默认开启**

### 3. 分配用户检查

- 分配检验单时，如果未选择分配用户会弹出确认提示
- 防止误操作导致检验单分配失败
- **强烈建议开启，默认开启**

### 4. 查看检验单链接

- 在业务受理页面可以直接查看检验单
- 点击物品名称或操作栏的"检验单"超链接即可跳转
- 无需切换页面，方便快捷
- **默认开启**

## 初验功能

### 1. 显示委托方英文名称

- 自动显示委托方的英文名称
- 方便填写国际单据
- **默认开启**

### 2. 自动设置初验内容

录入检验单时，插件会根据配置自动设置以下内容：

#### 可配置项：

- **分类**：化学品 / 锂电池类 / 器械类 / 磁性物质（默认：锂电池类）
- **结算方式**：月结（默认开启）
- **金额**：可自定义金额（默认：500.00）
- **下一年报告标记**：导入项目后自动标记为下一年报告（默认关闭）

### 3. 截图品名功能

- 双击品名可以快速截图并复制到剪贴板
- 方便制作报告时插入图片
- **默认关闭，需手动开启**

## 样品检验功能

### 1. 表单验证（核心功能）

强大的表单验证功能，支持锂电池和钠离子电池：

#### 验证内容包括：

- **品名验证**：检查中英文品名完整性、型号匹配
- **尺寸验证**：单位检查、形状匹配
- **净重计算**：自动计算并对比实际净重（误差超过5%提示）
- **瓦时数计算**：瓦时 = 电压 × 容量（误差超过5%提示）
- **能量密度检查**：超过 320Wh/kg 警告
- **包装说明验证**：根据不同包装说明检查必填项
- **危险电池型号检查**：对配置的危险型号进行警告
- **UN编号匹配**：自动验证UN编号是否正确
- **堆码测试规则**：根据包装说明检查是否需要堆码测试
- **跌落测试规则**：根据包装说明检查是否需要跌落测试

详细的验证规则请参见 [验证规则文档](./rule.md)

**默认开启**，可配置验证按钮位置（提交按钮旁边或右上角）

### 2. 附件解析与验证

::: tip 亮点
插件内置 WASM 附件解析引擎，无需安装额外软件！
:::

- **概要文档解析**：自动读取本地概要文件（.doc/.docx），提取关键信息
- **标签自动识别**：基于 YOLO 模型识别包装标签（9类、9A类、电池、货机标签）

**工作流程：**

```
通过 Everything 搜索本地附件
    ↓
WASM 引擎解析文件内容
    ↓
提取关键数据并与检验单对比
    ↓
显示验证结果
```

详细配置请参见 [附件验证文档](./attachment.md)

### 3. 数据复制功能

#### 项目编号复制

- 点击项目编号直接复制
- 或双击 `Ctrl` 键复制
- **默认开启**

#### 项目名称复制

- 双击项目名称复制
- 或按住 `Ctrl` 键并双击 `C` 键两次
- **默认开启**

### 4. 保存前提示

- 检验单修改后，标题前显示星号 `*`
- 关闭页面前弹出未保存提示
- 防止误操作导致数据丢失
- **默认开启**

### 5. 输入框修改标记

- 当输入框内容被修改后，改变背景颜色
- 直观显示哪些字段被修改过
- 可自定义背景颜色（默认：#76EEC6）
- **默认关闭**

### 6. 海陆运数据对比

- 在检验单页面点击表头复制表单数据到剪切板
- 在另一页面点击"对比"按钮进行数据对比
- 自动高亮显示差异项
- 方便核对空运和海运的数据差异
- **默认关闭**

### 7. 自定义主题

用更显眼的图标或字体颜色来区分不同运输方式：

- **空运（PEK）**：可自定义颜色（默认：#51a020 绿色）
- **海运（SEK）**：可自定义颜色（默认：#3e8ed0 蓝色）
- **道路（AEK）**：可自定义颜色（默认：#8c1af6 紫色）
- **铁路（REK）**：可自定义颜色（默认：#ea3323 红色）

**默认关闭，需手动开启**

### 8. 下一年报告主题

- 自定义下一年报告的字体颜色和背景颜色
- 方便区分当年和下一年的报告
- 字体颜色默认：空（使用系统默认）
- 背景颜色默认：#76EEC6（青绿色）

### 9. 新标签页打开

- 在新标签页打开网页链接
- 避免覆盖当前工作页面
- **默认关闭**

### 10. 自动刷新列表

- 离开页面后，返回时自动刷新列表
- 可配置刷新时间间隔（默认：10000毫秒 = 10秒）
- 保证数据实时性
- **默认开启**

## 样品检验 - 导入功能

### 1. 自动填充项目编号

- 从剪切板读取项目编号并自动填充
- 无需手动输入，提高效率
- **默认开启**

### 2. 自动填充分类

- 根据品名自动判断并填充物品分类
- 减少手动选择步骤
- **默认开启**

### 3. 自动设置检验单编号前缀

可根据运输方式自动设置检验单编号前缀：

- **空运（PEK）**：默认 PEKGZ2024
- **海运（SEK）**：默认 SEKGZ2024
- **道路（AEK）**：默认 AEKGZ2024
- **铁路（REK）**：默认 REKGZ2024

**默认关闭**，启用后需要手动配置各运输方式的前缀

## 样品检验单查询功能

### 自动填充查询项目编号

- 在查询窗口中自动填充项目编号
- 使用报告编号搜索时自动删除非法字符（如空格）
- 搜索报告时只需输入编号，无需输入日期范围
- **默认关闭**

## 注意事项

1. **验证规则持续更新**：验证规则会根据实际使用情况不断完善，欢迎反馈问题和建议
2. **配置灵活**：大部分功能都可以根据个人习惯开启或关闭
3. **兼容性**：支持 Chrome 系浏览器（Chrome、Edge、360浏览器等）和 Firefox 系浏览器
4. **数据安全**：所有操作都在本地进行，不会上传任何数据到第三方服务器
5. **危险电池提示**：可自定义危险电池型号列表，当检测到危险型号时会弹出警告
