🚀 一键操作

一键分配、一键回退、一键提交试验单，告别重复操作

⌨️ 快捷键支持

丰富的快捷键功能，包括 Ctrl+S 保存、Ctrl+D 刷新/导入。

✅ 智能验证

强大的表单验证功能，支持锂电池和钠离子电池。自动检查品名、尺寸、净重、瓦时数等 30+ 项规则。

📋 自动填充

根据配置自动设置初验内容、检验单编号前缀、物品分类等。从剪贴板智能读取项目编号，无需手动输入。

🎨 主题定制

自定义空运、海运、道路、铁路的主题颜色，用更显眼的图标和字体区分运输方式。可标记已修改的输入框，直观显示变更内容。

🔍 附件解析

自动读取本地图片概要，与检验单数据对比验证。标签自动检查。

📊 数据对比

支持数据对比功能，一键复制检验单数据，方便核对不同检验单的数据差异。

🛡️ 安全保护

未保存数据时关闭页面会弹出提示，标题显示星号标记。

🔄 自动刷新

离开页面一段时间后返回自动刷新[初验/检验单]列表。

📝 详细文档

提供完整的安装配置指南、功能介绍和验证规则说明。持续更新的文档，确保您充分利用所有功能。

🌐 跨浏览器支持

支持 Chrome、Edge、360 浏览器等 Chromium 内核浏览器，以及 Firefox 浏览器。提供针对不同浏览器的安装方式。

⚙️ 灵活配置

30+ 可配置项。

主要功能亮点 ​

业务受理 ​

  • ✅ 一键分配检验单给主检员
  • ✅ 一键提交试验单（支持12种试验类型）
  • ✅ 一键回退检验单状态
  • ✅ 直接查看检验单链接

初验功能 ​

  • ✅ 显示委托方英文名称
  • ✅ 自动设置分类、金额、结算方式
  • ✅ 支持标记下一年报告
  • ✅ 双击截图品名并复制

样品检验 ​

  • ✅ 核心功能：智能表单验证（30+ 验证规则）
  • ✅ 附件自动解析（图片概要、UN报告）
  • ✅ 标签和堆码评估单检查
  • ✅ 危险电池型号自动警告
  • ✅ 能量密度超限提示（>320Wh/kg）
  • ✅ 净重和瓦时数自动计算验证
  • ✅ UN 编号和包装说明匹配检查

便捷操作 ​

  • ✅ 项目编号和名称快速复制
  • ✅ 保存前防误关闭提示
  • ✅ 修改内容可视化标记
  • ✅ 空海运数据对比
  • ✅ 自动设置检验单编号前缀
  • ✅ 自动填充剪贴板项目编号
  • ✅ 根据品名自动填充分类
